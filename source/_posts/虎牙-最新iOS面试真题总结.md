## 核心知识点总结

| 主题         | 关键内容                               |
|--------------|--------------------------------------|
| 项目架构     | MVC/MVVM、模块化设计、性能优化       |

| RunLoop      | 事件监听、模式切换、定时器管理       |
| 网络协议     | HTTP/HTTPS、TCP三次握手、数据完整性  |
| 多线程       | GCD、NSOperation、线程安全           |

| 跨平台框架   | React Native 与 Flutter 对比          |
